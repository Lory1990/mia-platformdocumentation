#Overview

![image alt text](img/development-suite.png)

The suite aims to create and release the platform in a few clicks. It has been designed to simplify the work of developer and consists in three main components:

* **API CONSOLE:** it allows to configure and manage the platform

* **API PORTAL:** it allows to document the API REST created with the API Console

* **MONITORING DASHBOARD:** it allow to monitoring the services created with the API
